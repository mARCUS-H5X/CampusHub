# CampusFlow 🎓

A premium, interactive campus dashboard homepage built with vanilla HTML, CSS, and JavaScript. Gaming-inspired dark theme with glassmorphism, neon accents, and smooth animations.

## Features

- **Collapsible Sidebar** — Smooth expand/collapse with icon & label modes
- **Hero Announcements** — Dynamic filtering by branch & semester, important-first sorting
- **Upcoming Events** — Horizontal scroller with Register/Interested button states
- **Campus Resources** — Filterable resource list with download actions
- **Lost & Found** — Grid layout with status badges and report functionality  
- **Campus Feed** — Post cards with filter tabs (All/Important/Events/Help), like/comment/share
- **Right Sidebar Widgets** — Active Channels, Productivity Goals, Trending Tags, Campus Poll
- **Animations** — Page-load fade-ins, parallax, hover effects, toast notifications

## Tech Stack

- HTML5 (semantic)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (no frameworks)

## Getting Started

```bash
# Clone the repo
git clone https://github.com/mARCUS-H5X/CampusHub.git

# Open in browser
open index.html
```

Or use **Live Server** in VS Code for the best experience.
